 /service This is the service used to pass to the service node the user choice inserted.

Description\+:

The node is used to manage all the UI (User Interface) actions. This is a way to have them divided from the normal execution of the program. In this way there is no possibility that the program waits for an input and so blocks the other processes to go on with the normal running.
